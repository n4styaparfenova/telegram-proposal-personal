# Предложение о сотрудничестве — Личная версия

Личное предложение от Даниила Пономарева о создании совместного миллионного Telegram-канала.

## Описание

Это более личная и неформальная версия коммерческого предложения, написанная от первого лица. Подходит для личных встреч и неформального общения с блогерами.

## Основные отличия от корпоративной версии

- Тексты от первого лица ("Я", "Моя идея")
- Более живой и дружелюбный тон
- Добавлен блок "Немного обо мне"
- Упрощённые формулировки
- Убрана корпоративная стилистика
- Акцент на личном опыте и доверии

## Файлы

- `index.html` - основной файл лендинга

## Как открыть

Откройте `index.html` в любом браузере.
